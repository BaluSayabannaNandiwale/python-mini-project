# QR Code Generator

## Description

This project provides a simple Python script to generate a QR code for a given URL using the `qrcode` library.

## Features

- **QR Code Generation**: Generates a QR code for a specified URL.
- **Image Saving**: Saves the generated QR code as an image file.

## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/your-username/qr-code-generator.git
    cd qr-code-generator
    ```

2. Install the required package:
    ```sh
    pip install qrcode
    ```

## Usage

1. Open the `qr_code_generator.py` file.
2. Modify the URL in the script as needed.
3. Run the script:
    ```sh
    python qr_code_generator.py
    ```

4. The script will generate a QR code for the specified URL and save it as an image file named `wscube.png`.

## Code Explanation

- **QR Code Generation**: The script uses the `qrcode` library to generate a QR code for the specified URL.
- **Image Saving**: The generated QR code is saved as an image file.
